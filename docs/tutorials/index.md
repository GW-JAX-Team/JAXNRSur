# Tutorials

- **[Getting Started](getting_started.ipynb)** — Model instantiation, waveform evaluation in geometric and physical units, automatic differentiation, and batch computation with `vmap`.

- **[Frequency-Domain Waveforms](frequency_domain.ipynb)** — Obtaining time- and frequency-domain polarizations in physical units using the `JAXNRSur` wrapper.
